# 3D Logistics Simulations

Combined React + Three.js application containing:

1. Operational Supply Chain Digital Twin
2. Forecast-to-Transport Capacity Simulation

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Node.js: 24.x
- Install Command: leave on automatic detection, or use `npm install --no-audit --no-fund`
